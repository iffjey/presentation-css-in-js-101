Tooling

  • Linting - stylelint-processor-styled-components
  • Syntax highlighting - vscode-styled-components
  • Snapshot testing - jest-styled-components
  • Mixins and helpers - polished ${ ‚ú® }
  • Design system utilities - styled-system
  • Autocomplete - css-in-js: Extension for Atom and VS code

Drawbacks

  • 0.2% of browsers have javascript disabled
  • Tooling: still not perfect (e.g. autocomplete doesn't support template literals)
  • User inputs must be sanitized - How can I use CSS-in-JS securely?
  • Absence of atomic web design leads to "spaghetti code" - Atomic Web Design
